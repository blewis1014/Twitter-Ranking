      function pianoReadCustomVariableCookie(specificName) {
          var cookieValue;
          try {
              // Try to get the cookie value with regexp
              var match = document.cookie.match(new RegExp('(^| )__pcvc=([^;]+)'));
              if (match) {
                  // If cookie with this name was found - try to parse the JSON value
                  cookieValue = JSON.parse(match[2]);
              }
          }
          catch (e) {
              // By default - set value as empty object
              cookieValue = {};
          }
          if (!cookieValue) {
              // By default - set value as empty object
              cookieValue = {};
          }
          if (typeof specificName != "undefined") {
              // Get the specific value from the set
              if (typeof cookieValue[specificName] != "undefined") {
                  return cookieValue[specificName];
              }
              return null;
          }
          return cookieValue;
      }



      function pianoSetCustomVariableCookie(name, value) {
          // Get all existing values from the cookie
          var cookieValue = pianoReadCustomVariableCookie();
          // Set the value
          cookieValue[name] = value;

          // Get cookie expiration date in 3 years
          var d = new Date();
          d.setTime(d.getTime() + (94608000000));
          var expires = "expires=" + d.toUTCString();

          // Write the cookie value
          document.cookie = "__pcvc=" + JSON.stringify(cookieValue) + ";" + expires + ";path=/";
      }

                  <div class="bigtext">

                    
                      
                    

                    

                    
                      

                      
                        
                        

                        

                      

                      <p><a href="/topics/hunter-biden/">Hunter Biden</a> was receiving a $10 million annual fee from a Chinese billionaire who has been accused of corruption and with whom he sought to increase the cash flow with a joint business ownership, an August 2017 email from Mr. <a href="/topics/joseph-r-biden/">Biden</a> shows.</p>

                      

                      

                      

                    
                      

                      

                      <p>The money was for &#8220;introductions alone,&#8221; said Mr. <a href="/topics/joseph-r-biden/">Biden</a>, son of Democratic presidential nominee <a href="/topics/joseph-r-biden/">Joseph R. Biden</a>. Another email to <a href="/topics/hunter-biden/">Hunter Biden</a> from a consultant says that a 10% stake in an unnamed company will go to &#8220;the big guy?&#8221; The &#8220;big guy&#8221; was not identified.</p>

                      

                      

                      

                    
                      

                      

                      <p><a href="/topics/hunter-biden/">Hunter Biden</a> hooked up with the billionaire, <a href="/topics/ye-jianming/">Ye Jianming</a>, chairman of the CEFC China Energy Co. conglomerate, during his father&#8217;s vice presidency.</p>

                      

                        <div class="article-toplinks">

                          
                          
                          
                          
                          
                          
                          
                          
                          
                          
                          
                          
                          

                          
                            <aside>
                            <hr/>
                            <p><strong>
                              
                                <a href="/news/2020/oct/15/trump-campaign-says-twitter-suspended-its-official/" target="_blank">
                              
                              
                                SEE ALSO: Trump campaign says Twitter suspended its official campaign account
                              
                              </a></strong></p>
                            <hr style="margin-bottom: 15px;"/>
                            </aside>
                          
                        </div>

                      

                      

                      

                    
                      

                      

                      <p>The emails were first reported by The New York Post on Thursday.</p>

                      

                      

                      

                    
                      

                      

                      <p>The newspaper obtained a cache of messages, documents and photos from a MacBrook Pro laptop purportedly owned by <a href="/topics/hunter-biden/">Hunter Biden</a>, via President Trump&#8217;s personal attorney, Rudolph W. Giuliani.</p>

                      

                      
                      <div class="connatixcontainer">
                        <hr/>
                        <script id="52b1f7f094294ef9a64f6c534558cada"> cnxps.cmd.push(function () { cnxps({ playerId: '2dd9afad-0104-402b-b341-830f7d9e8ccc' }).render('52b1f7f094294ef9a64f6c534558cada'); }); </script>
                        <hr/>
                      </div>
                      

                      

                    
                      

                      

                      <p>Democratic candidate Mr. <a href="/topics/hunter-biden/">Biden</a> has denied any wrongdoing.</p>

                      

                      

                      
                        
                        
                        
                        
                        
                        
                        
                        
                        
                        
                        
                        

                        

                      

                    
                      

                      

                      <p>His campaign said investigations into Mr. Biden and Hunter have &#8220;reached the same conclusion: that Joe Biden carried out official U.S. policy toward <a href="/topics/ukraine/">Ukraine</a> and engaged in no wrongdoing.&#8221;</p>

                      

                      

                      

                    
                      

                      

                      <p>The New York Post stories have helped keep the allegations alive less than three weeks from Election Day.</p>

                      

                      

                      

                    
                      

                      

                      <p>Mr. <a href="/topics/hunter-biden/">Biden</a> told donors Thursday that Mr. Trump is ratcheting up a &#8220;misinformation&#8221; campaign.</p>

                      

                      

                      

                    
                      
                        <a name="pagebreak"></a>
                      

                      

                      <p>&#8220;We have 19 days left, and you know, he&#8217;s going to throw everything but the kitchen sink at me,&#8221; Mr. <a href="/topics/hunter-biden/">Biden</a> said.</p>

                      

                      

                      

                    

                    
                      <p>Mr. Giuliani, meanwhile, blasted out a screenshot Thursday of a text he claims <a href="/topics/hunter-biden/">Hunter Biden</a> sent to his daughter, Naomi Biden, on Jan. 3, 2019.</p>
                    
                      <p>&#8220;I Hope you all cal do what I did and pay for everything for this entire family Fro 30 years,&#8221; the text reads. &#8220;It&#8217;s really hard. But don&#8217;t worry unlike Pop I won&#8217;t make you give me half your salary.&#8221;</p>
                    
                      <p>Mr. Giuliani said, &#8220;This describes the amount kicked back to the Boss, Joe Biden, as happens in every organized crime group.&#8221;</p>
                    
                      <p>Mr. Giuliani also said <a href="/topics/hunter-biden/">Hunter Biden</a> was &#8220;heavily inebriated&#8221; when he dropped off his laptop at a Delaware repair shop in April 2019.</p>
                    
                      <p>The owner of the shop, John Paul Mac Isaac, claimed legal ownership since nobody picked it up, and he provided a download to the FBI and to Mr. Giuliani, who was conducting an investigation into <a href="/topics/hunter-biden/">Hunter Biden</a>&#8217;s financial ties to an energy business in <a href="/topics/ukraine/">Ukraine</a>.</p>
                    
                      <p>The FBI and U.S. attorney in Delaware refused to comment on anything related to the laptop, including whether they were in possession of it, citing their policy of neither confirming nor denying the existence of an investigation.</p>
                    
                      <p>A report by Senate Republicans issued last month said <a href="/topics/hunter-biden/">Hunter Biden</a> &#8220;cashed in&#8221; on his father&#8217;s vice presidency. The report said the Biden-Ye business relationship intensified in 2015. <a href="/topics/hunter-biden/">Hunter Biden</a> flew with his father, then the vice president, on Air Force 2 to China in 2013 and made business contacts there.</p>
                    
                      <p>Mr. Ye, 43, who maintained close ties to the Chinese military and Communist Party, was accused by the government of corruption and vanished in 2018.</p>
                    
                      <p><a href="/topics/hunter-biden/">Hunter Biden</a>&#8217;s 2017 email was sent to Gongwen Dong, a Ye confident who brokered his huge investment deals and dealt directly with <a href="/topics/hunter-biden/">Hunter Biden</a>.</p>
                    
                      <p><a href="/topics/hunter-biden/">Hunter Biden</a> laid out his expectations for more money from the &#8220;chairman&#8221; in the August 2017 message.</p>
                    
                      <p>&#8220;My understanding is that the original agreement with the Director was for consulting fees based on introductions alone a rate of $10 million per year for for three years guarantee total of $30 million,&#8221; <a href="/topics/hunter-biden/">Hunter Biden</a> wrote to Mr. Gongwen. &#8220;The chairman [Mr. Ye] changed that deal after we me[t] in Miami TO A MUCH MORE LASTING AND LUCRATIVE ARRANGEMENT to create a holding company 50% percent owned by ME and 50% owned by him. Consulting fees is one piece of our income stream but the reason this proposal by the chairman was so much more interesting to me and my family is that we would also be partners in the equity and profits of the JV [joint venture].&#8221;</p>
                    
                      <p><a href="/topics/hunter-biden/">Hunter Biden</a> told Mr. Gongwen in the email that if the joint venture did not work, &#8220;then please return us to the original deal 10M per year a guaranteed 3 years plus bonus payments for any successful deal we introduce.&#8221;</p>
                    
                      <p>The New York Post reproduced a second email Thursday showing an international consultant, James Gilliar, telling <a href="/topics/hunter-biden/">Hunter Biden</a> in May 2017 about how Ye Jianming&#8217;s CEFC will divide up payments for an unnamed company for which Hunter is &#8220;chair/vice chair.&#8221;</p>
                    
                      <p>The email says Hunter will receive a 20% equity, plus a 10% stake &#8220;held by H for the big guy?&#8221;</p>
                    
                      <p>The email does not identify the &#8220;big guy.&#8221;</p>
                    
                      <p>In a conference call organized by the Trump campaign, Rep. Jim Banks of Indiana said <a href="/topics/hunter-biden/">Hunter Biden</a> is lucky not to be in jail, and said the former vice president has some answering to do.</p>
                    
                      <p>&#8220;What we want to know today is Joe Biden the big guy?&#8221; Mr. Banks said. &#8220;When there is discussion of Hunter facilitation &#8216;introductions&#8217; for $10 million who is he expected to introduce to whom? Are they talking about Joe Biden?&#8221;</p>
                    
                      <p>The New York Post reported on Wednesday that a top executive at <a href="/topics/ukraine/">Ukraine</a>&#8217;s Burisma Holdings energy company sent <a href="/topics/hunter-biden/">Hunter Biden</a> an email thanking him for setting up a 2015 meeting with his father, then the vice president. That narrative flies in the face of the elder <a href="/topics/hunter-biden/">Biden</a>&#8217;s claims that he had no knowledge of his son&#8217;s business deals.</p>
                    
                      <p>President Barack Obama named Mr. <a href="/topics/hunter-biden/">Biden</a> as the point man on <a href="/topics/ukraine/">Ukraine</a> in February 2014. Three months later, <a href="/topics/hunter-biden/">Hunter Biden</a> was added to the Burisma board and raked in millions of dollars in the ensuing years, the Senate report said.</p>
                    
                      <p>The <a href="/topics/hunter-biden/">Biden</a> campaign responded by saying there is no official record of such a meeting but that there could have been a brief encounter, Politico reported.</p>
                    
                      <p><a href="/topics/hunter-biden/">Hunter Biden</a>&#8217;s attorney has not denied the emails&#8217; authenticity.</p>
                    
                      <p>Senate Homeland Security and Government Affairs Chairman Ron Johnson, Wisconsin Republican, said on Fox News that a &#8220;whistleblower&#8221; &mdash; apparently computer shop owner Mr. Isaac &mdash; has delivered data to the committee, which will be evaluated.</p>
                    
                      <p>He and Senate Finance Committee Chairman Charles E. Grassley, Iowa Republican, issued a report on <a href="/topics/hunter-biden/">Hunter Biden</a>&#8217;s far-flung financial ties during the Obama administration. Besides China and <a href="/topics/ukraine/">Ukraine</a>, Mr. <a href="/topics/hunter-biden/">Biden</a> received a $3.5 million wire transfer from a Russian woman oligarch in February 2014 for consulting services. He also received millions of dollars in payments from Burisma as a board member.</p>
                    
                      <p>The Senate report details how <a href="/topics/hunter-biden/">Hunter Biden</a> set up an LLC called Hudson West in April 2016 that became a receiving point for Chinese cash. Mr. Gongwen, Mr. <a href="/topics/ye-jianming/">Ye</a>&#8217;s associate, played a role in setting up the firm.</p>
                    
                      <p>In August 2017, the same month <a href="/topics/hunter-biden/">Hunter Biden</a> was talking to Mr. Gongwen about his $10 million retainer, CEFC wired Hudson West $5 million. Hunter <a href="/topics/hunter-biden/">Biden</a> sent $4.7 million to his law firm, the Senate report shows.</p>

      function pianoReadCustomVariableCookie(specificName) {
          var cookieValue;
          try {
              // Try to get the cookie value with regexp
              var match = document.cookie.match(new RegExp('(^| )__pcvc=([^;]+)'));
              if (match) {
                  // If cookie with this name was found - try to parse the JSON value
                  cookieValue = JSON.parse(match[2]);
              }
          }
          catch (e) {
              // By default - set value as empty object
              cookieValue = {};
          }
          if (!cookieValue) {
              // By default - set value as empty object
              cookieValue = {};
          }
          if (typeof specificName != "undefined") {
              // Get the specific value from the set
              if (typeof cookieValue[specificName] != "undefined") {
                  return cookieValue[specificName];
              }
              return null;
          }
          return cookieValue;
      }



      function pianoSetCustomVariableCookie(name, value) {
          // Get all existing values from the cookie
          var cookieValue = pianoReadCustomVariableCookie();
          // Set the value
          cookieValue[name] = value;

          // Get cookie expiration date in 3 years
          var d = new Date();
          d.setTime(d.getTime() + (94608000000));
          var expires = "expires=" + d.toUTCString();

          // Write the cookie value
          document.cookie = "__pcvc=" + JSON.stringify(cookieValue) + ";" + expires + ";path=/";
      }

                  <div class="bigtext">

                    
                      
                    

                    

                    
                      

                      
                        
                        

                        

                      

                      <p><a href="/topics/hunter-biden/">Hunter Biden</a> was receiving a $10 million annual fee from a Chinese billionaire who has been accused of corruption and with whom he sought to increase the cash flow with a joint business ownership, an August 2017 email from Mr. <a href="/topics/joseph-r-biden/">Biden</a> shows.</p>

                      

                      

                      

                    
                      

                      

                      <p>The money was for &#8220;introductions alone,&#8221; said Mr. <a href="/topics/joseph-r-biden/">Biden</a>, son of Democratic presidential nominee <a href="/topics/joseph-r-biden/">Joseph R. Biden</a>. Another email to <a href="/topics/hunter-biden/">Hunter Biden</a> from a consultant says that a 10% stake in an unnamed company will go to &#8220;the big guy?&#8221; The &#8220;big guy&#8221; was not identified.</p>

                      

                      

                      

                    
                      

                      

                      <p><a href="/topics/hunter-biden/">Hunter Biden</a> hooked up with the billionaire, <a href="/topics/ye-jianming/">Ye Jianming</a>, chairman of the CEFC China Energy Co. conglomerate, during his father&#8217;s vice presidency.</p>

                      

                        <div class="article-toplinks">

                              </a></strong></p>
                            <hr style="margin-bottom: 15px;"/>

                      <p>The emails were first reported by The New York Post on Thursday.</p>

                      

                      

                      

                    
                      

                      

                      <p>The newspaper obtained a cache of messages, documents and photos from a MacBrook Pro laptop purportedly owned by <a href="/topics/hunter-biden/">Hunter Biden</a>, via President Trump&#8217;s personal attorney, Rudolph W. Giuliani.</p>

                      

                      
                      <div class="connatixcontainer">
                        <hr/>
                        <script id="52b1f7f094294ef9a64f6c534558cada"> cnxps.cmd.push(function () { cnxps({ playerId: '2dd9afad-0104-402b-b341-830f7d9e8ccc' }).render('52b1f7f094294ef9a64f6c534558cada'); }); </script>

                      <p>Democratic candidate Mr. <a href="/topics/hunter-biden/">Biden</a> has denied any wrongdoing.</p>

                      

                      

                      
                        
                        
                        
                        
                        
                        
                        
                        
                        
                        
                        
                        

                        

                      

                    
                      

                      

                      <p>His campaign said investigations into Mr. Biden and Hunter have &#8220;reached the same conclusion: that Joe Biden carried out official U.S. policy toward <a href="/topics/ukraine/">Ukraine</a> and engaged in no wrongdoing.&#8221;</p>

                      

                      

                      

                    
                      

                      

                      <p>The New York Post stories have helped keep the allegations alive less than three weeks from Election Day.</p>

                      

                      

                      

                    
                      

                      

                      <p>Mr. <a href="/topics/hunter-biden/">Biden</a> told donors Thursday that Mr. Trump is ratcheting up a &#8220;misinformation&#8221; campaign.</p>

                      

                      

                      

                    
                      
                        <a name="pagebreak"></a>
                      

                      

                      <p>&#8220;We have 19 days left, and you know, he&#8217;s going to throw everything but the kitchen sink at me,&#8221; Mr. <a href="/topics/hunter-biden/">Biden</a> said.</p>

                      

                      

                      

                    

                    
                      <p>Mr. Giuliani, meanwhile, blasted out a screenshot Thursday of a text he claims <a href="/topics/hunter-biden/">Hunter Biden</a> sent to his daughter, Naomi Biden, on Jan. 3, 2019.</p>
                    
                      <p>&#8220;I Hope you all cal do what I did and pay for everything for this entire family Fro 30 years,&#8221; the text reads. &#8220;It&#8217;s really hard. But don&#8217;t worry unlike Pop I won&#8217;t make you give me half your salary.&#8221;</p>
                    
                      <p>Mr. Giuliani said, &#8220;This describes the amount kicked back to the Boss, Joe Biden, as happens in every organized crime group.&#8221;</p>
                    
                      <p>Mr. Giuliani also said <a href="/topics/hunter-biden/">Hunter Biden</a> was &#8220;heavily inebriated&#8221; when he dropped off his laptop at a Delaware repair shop in April 2019.</p>
                    
                      <p>The owner of the shop, John Paul Mac Isaac, claimed legal ownership since nobody picked it up, and he provided a download to the FBI and to Mr. Giuliani, who was conducting an investigation into <a href="/topics/hunter-biden/">Hunter Biden</a>&#8217;s financial ties to an energy business in <a href="/topics/ukraine/">Ukraine</a>.</p>
                    
                      <p>The FBI and U.S. attorney in Delaware refused to comment on anything related to the laptop, including whether they were in possession of it, citing their policy of neither confirming nor denying the existence of an investigation.</p>
                    
                      <p>A report by Senate Republicans issued last month said <a href="/topics/hunter-biden/">Hunter Biden</a> &#8220;cashed in&#8221; on his father&#8217;s vice presidency. The report said the Biden-Ye business relationship intensified in 2015. <a href="/topics/hunter-biden/">Hunter Biden</a> flew with his father, then the vice president, on Air Force 2 to China in 2013 and made business contacts there.</p>
                    
                      <p>Mr. Ye, 43, who maintained close ties to the Chinese military and Communist Party, was accused by the government of corruption and vanished in 2018.</p>
                    
                      <p><a href="/topics/hunter-biden/">Hunter Biden</a>&#8217;s 2017 email was sent to Gongwen Dong, a Ye confident who brokered his huge investment deals and dealt directly with <a href="/topics/hunter-biden/">Hunter Biden</a>.</p>
                    
                      <p><a href="/topics/hunter-biden/">Hunter Biden</a> laid out his expectations for more money from the &#8220;chairman&#8221; in the August 2017 message.</p>
                    
                      <p>&#8220;My understanding is that the original agreement with the Director was for consulting fees based on introductions alone a rate of $10 million per year for for three years guarantee total of $30 million,&#8221; <a href="/topics/hunter-biden/">Hunter Biden</a> wrote to Mr. Gongwen. &#8220;The chairman [Mr. Ye] changed that deal after we me[t] in Miami TO A MUCH MORE LASTING AND LUCRATIVE ARRANGEMENT to create a holding company 50% percent owned by ME and 50% owned by him. Consulting fees is one piece of our income stream but the reason this proposal by the chairman was so much more interesting to me and my family is that we would also be partners in the equity and profits of the JV [joint venture].&#8221;</p>
                    
                      <p><a href="/topics/hunter-biden/">Hunter Biden</a> told Mr. Gongwen in the email that if the joint venture did not work, &#8220;then please return us to the original deal 10M per year a guaranteed 3 years plus bonus payments for any successful deal we introduce.&#8221;</p>
                    
                      <p>The New York Post reproduced a second email Thursday showing an international consultant, James Gilliar, telling <a href="/topics/hunter-biden/">Hunter Biden</a> in May 2017 about how Ye Jianming&#8217;s CEFC will divide up payments for an unnamed company for which Hunter is &#8220;chair/vice chair.&#8221;</p>
                    
                      <p>The email says Hunter will receive a 20% equity, plus a 10% stake &#8220;held by H for the big guy?&#8221;</p>
                    
                      <p>The email does not identify the &#8220;big guy.&#8221;</p>
                    
                      <p>In a conference call organized by the Trump campaign, Rep. Jim Banks of Indiana said <a href="/topics/hunter-biden/">Hunter Biden</a> is lucky not to be in jail, and said the former vice president has some answering to do.</p>
                    
                      <p>&#8220;What we want to know today is Joe Biden the big guy?&#8221; Mr. Banks said. &#8220;When there is discussion of Hunter facilitation &#8216;introductions&#8217; for $10 million who is he expected to introduce to whom? Are they talking about Joe Biden?&#8221;</p>
                    
                      <p>The New York Post reported on Wednesday that a top executive at <a href="/topics/ukraine/">Ukraine</a>&#8217;s Burisma Holdings energy company sent <a href="/topics/hunter-biden/">Hunter Biden</a> an email thanking him for setting up a 2015 meeting with his father, then the vice president. That narrative flies in the face of the elder <a href="/topics/hunter-biden/">Biden</a>&#8217;s claims that he had no knowledge of his son&#8217;s business deals.</p>
                    
                      <p>President Barack Obama named Mr. <a href="/topics/hunter-biden/">Biden</a> as the point man on <a href="/topics/ukraine/">Ukraine</a> in February 2014. Three months later, <a href="/topics/hunter-biden/">Hunter Biden</a> was added to the Burisma board and raked in millions of dollars in the ensuing years, the Senate report said.</p>
                    
                      <p>The <a href="/topics/hunter-biden/">Biden</a> campaign responded by saying there is no official record of such a meeting but that there could have been a brief encounter, Politico reported.</p>
                    
                      <p><a href="/topics/hunter-biden/">Hunter Biden</a>&#8217;s attorney has not denied the emails&#8217; authenticity.</p>
                    
                      <p>Senate Homeland Security and Government Affairs Chairman Ron Johnson, Wisconsin Republican, said on Fox News that a &#8220;whistleblower&#8221; &mdash; apparently computer shop owner Mr. Isaac &mdash; has delivered data to the committee, which will be evaluated.</p>
                    
                      <p>He and Senate Finance Committee Chairman Charles E. Grassley, Iowa Republican, issued a report on <a href="/topics/hunter-biden/">Hunter Biden</a>&#8217;s far-flung financial ties during the Obama administration. Besides China and <a href="/topics/ukraine/">Ukraine</a>, Mr. <a href="/topics/hunter-biden/">Biden</a> received a $3.5 million wire transfer from a Russian woman oligarch in February 2014 for consulting services. He also received millions of dollars in payments from Burisma as a board member.</p>
                    
                      <p>The Senate report details how <a href="/topics/hunter-biden/">Hunter Biden</a> set up an LLC called Hudson West in April 2016 that became a receiving point for Chinese cash. Mr. Gongwen, Mr. <a href="/topics/ye-jianming/">Ye</a>&#8217;s associate, played a role in setting up the firm.</p>
                    
                      <p>In August 2017, the same month <a href="/topics/hunter-biden/">Hunter Biden</a> was talking to Mr. Gongwen about his $10 million retainer, CEFC wired Hudson West $5 million. Hunter <a href="/topics/hunter-biden/">Biden</a> sent $4.7 million to his law firm, the Senate report shows.</p>
                    
                      <p><em>&bull; Seth McLaughlin and Jeff Mordock contributed to this report.</em></p>
